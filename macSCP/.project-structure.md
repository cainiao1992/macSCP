# macSCP Project Structure

## Proposed Organization

```
macSCP/
├── Models/
│   ├── SSHConnection.swift
│   ├── ConnectionFolder.swift
│   ├── RemoteFile.swift
│   └── Item.swift
│
├── Managers/
│   ├── CitadelSFTPManager.swift
│   ├── SFTPManager.swift (legacy)
│   ├── SSHManager.swift (legacy)
│   └── RemoteClipboard.swift
│
├── Views/
│   ├── Main/
│   │   ├── ContentView.swift
│   │   ├── ConnectionListView.swift
│   │   └── SSHConnectionView.swift
│   │
│   ├── Windows/
│   │   ├── SSHFileExplorerWindow.swift
│   │   ├── FileEditorView.swift
│   │   └── FileInfoView.swift
│   │
│   └── Components/
│       ├── FinderStyleBrowserView.swift
│       ├── RemoteFileBrowserView.swift
│       ├── PasswordPromptView.swift
│       └── FinderComponents/
│           ├── FinderFileRow.swift
│           ├── BreadcrumbView.swift
│           ├── ClipboardStatusView.swift
│           ├── LoadingOverlay.swift
│           └── EmptyStateView.swift
│
├── Assets.xcassets/
└── macSCPApp.swift
```

## Benefits:
- Clear separation of concerns
- Easier navigation and maintenance
- Better scalability for future features
- Follows SwiftUI best practices
